Documentation in Spanish (on Apertium 1.0), Catalan (on Apertium 2.0) and English (to be updated for Apertium 2.0) is available through the sf.net/projects/apertium project page.

The following documents, also available in the Docs section of the Apertium SourceForge project may be useful (they refer mainly to Apertium 1.0):

A wiki has also been set up for users and developers to generate additional documentation
