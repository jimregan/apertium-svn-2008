Que é Apertium?

Apertium é unha plataforma de tradución automática, inicialmente pensada para pares de linguas emparentadas, pero que foi recentemente expandida para poder tratar pares de linguas máis diverxentes (por exemplo inglés–catalán). A plataforma proporciona (a) un motor de tradución independente das linguas, (b) ferramentas para xestionar os datos lingüísticos necesarios para construír un sistema de tradución automática para un par de linguas dado e (c) datos lingüísticos para un número crecente de pares de linguas.

Apertium usa un motor de tradución de transferencia superficial que procesa o texto de entrada en etapas, como nunha cadea de montaxe: desformateo, análise morfolóxica, desambiguación categorial, transferencia estructural superficial, transferencia léxica, xeración morfolóxica e reformateo.

Apertium usa transductores de estados finitos para as operacións de procesamento léxico (análises e xeración morfolóxica, transferencia léxica), modelos ocultos de Markov para a desambiguación categorial e chunking multietapa baseado en estados finitos para a transferencia superficial.

O deseño inicial está baseado no de sistemas que xa desenvolvera o grupo Transducens da Universitat d'Alacant, por exemplo interNOSTRUM (español–catalán), e Traductor Universia (español–portugués).

É posíbel usar Apertium para construír sistemas de tradución automática para unha gran variedade de pares de linguas; para iso, Apertium usa formatos sinxelos baseados en XML para codificar os datos lingüísticos necesarios (creados a man ou por conversión de datos existentes) que se compilan, coas ferramentas que se provén, aos formatos de gran velocidade usados polo motor.
